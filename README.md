# Decidash TS SDK

## Package Install
```
pnpm add @coldbell/decidash-ts-sdk
```

## Install & Build
```bash
pnpm install
pnpm build
```

## Run Example
```bash
npx tsx example/account.ts
npx tsx example/contract.ts
npx tsx example/market.ts
npx tsx example/websocket.ts
npx tsx example/traders.ts
```
